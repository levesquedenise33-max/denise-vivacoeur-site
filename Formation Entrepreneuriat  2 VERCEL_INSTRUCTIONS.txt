✅ ÉTAPE 3: UPLOAD LES FICHIERS SUR GITHUB
═══════════════════════════════════════════════════════════════════════════

Dans ton repository GitHub:

1. Clique "Add file" > "Upload files"

2. Drag & drop:
   - index.html
   - vivacoeur-ia-app.html
   - ViVaCoeur_IA_Guide_Complet.docx
   - vivacoeur-ia-deployment.md
   - README.md
   - EMAIL_ETUDIANTS.txt

3. Commit changes

═══════════════════════════════════════════════════════════════════════════
✅ ÉTAPE 4: CONNECTE À VERCEL
═══════════════════════════════════════════════════════════════════════════

1. Va sur dashboard Vercel: https://vercel.com/dashboard

2. Clique "New Project"

3. Trouve ton repository: denise-vivacoeur-site

4. Clique "Import"

5. Settings:
   Framework: Other (ou HTML)
   Root Directory: ./
   
6. Clique "Deploy"

✅ BOOM! Ton site est EN LIGNE!
